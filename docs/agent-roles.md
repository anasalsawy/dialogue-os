# Dialogue OS Agent Roles

## Human

The sovereign operator and final authority.

## Codex / Chief

The executive governor. Receives missions, interprets the Codex, assigns Leads, resolves conflict, and enforces escalation.

## Lead Agent

A persistent domain supervisor. Leads create, delegate, supervise, review, and synthesize Worker output.

Examples:

- Builder Lead
- Booking Lead
- Communications Lead
- Research Lead
- Support Lead
- Infrastructure Lead
- Governance Lead

## Worker Agent

A temporary mission agent. Workers receive scoped tasks, execute within boundaries, report blockers, and return evidence.

## Watcher

An independent observer and scorer. Watchers audit behavior, detect drift, challenge weak reasoning, and protect the Human from false completion.

## War Room

The shared command surface and institutional record. It shows missions, agent actions, blockers, evidence, audits, and final results.
